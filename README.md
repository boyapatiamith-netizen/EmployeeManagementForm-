Employee Management Form
. Description
This is a simple Java Swing application that allows users to manage employee information.
The program lets you add employees by entering their name, department, and salary, and then displays the list in a table below.
It demonstrates the use of:
JFrame, JPanel, and JTable for GUI layout
DefaultTableModel for handling table data
Event handling with ActionListener
Basic input validation
.Features
 Add new employees with their details
Automatically updates the employee table
 Validates user input (salary must be a number)
 Clears input fields after successful addition
 Technologies Used
Java (JDK 8 or later)
Swing (GUI components)
JTable and DefaultTableModel
